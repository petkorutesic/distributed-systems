ssh (Secure Shell)

  • cryptographic network protocol
  • used for operating network services securely over unsecured network
  • important to cloud computing
    • avoiding exposing a cloud-based virtual machines directly on the internet
  • Typically used for:
    • romote command-login and execution
  • standart TCP port for SSH is 22
  • ssh sends encrypted messages and doesn't disclose passwords in plaintext
  • however files could be maybe decrypted by NSA as Snowden's case shows (source Wikipedia)

    • ssh - generating keys

    • ssh-keygen
    • populating directory with public/private key pair
    • id_rsa - file in which private key is by default stored
    • id_rsa.pub - file in which public key is stored
    • private key is never transferred through the network during authentication
    • SSH has to ensure that the person offering the public key also owns the private key

    using ssh

    • The command syntax is as follows:
      • ssh username@ip.address
    • In our case it will be, for instance:
      • ssh ubuntu@142.72.192.31

    SCP(Secure copy protocol)

    • used to copy files and folders from one machine to another using SSH
    • main syntax
      • scp username:ip_addres_a:/path/to/file username@ip_addres_b:/path/to/destination
    • syntax for transferring file from local machine to remote machine
      • scp /path/to/file username@10.0.0.165:/home/username/

    ICMP (Internet control message protocol)

    • error-reporting protocol
    • supporting protocol in the Internet protocol suite (TCP/IP)
    • ping and traceroute uses ICMP packets
    • all network-connected devices can process ICMP messages

    Computer networking utilities

    • ping ping tests the reachability of a host
      • ping 10.0.0.1
      • ping uses ICMP packets
    • netcat - useful tool for scanning remote system ports
      • uses TCP or UDP protocols
      • examples
        • nc google.com 443
        • nc 10.0.0.2 22
      • can be used to perform http requests:
      • printf "GET /hypertext/WWW/TheProject.html HTTP/1.0\r\nHost: info.cern.ch\r\n\r\n" | nc info.cern.ch 80nc 10.0.0.2 22
    • nmap - useful tool used by system and network administrators
      • nmap -p 80 -sT google.com

    FTP (File Transfer Protocol)

  • uses two separate TCP connectons (control and data connections)
  • way of exchanging files between client and server
  • FTP clients provide easy to use graphics user interface

    • Examples of FTP applications

    • Mac
      • CyberDuck, FileZilla
    • Windows
      • WinSCP, Free FTP
    • Linux
      • FileZilla, CrossFTP
